2.Descriere generala

2.1 Descrierea Produsului

Aplicatia ZooPlanet, bazata pe proiectul MeoW, este developata pentru persoanele care vor sa caute informatii despre animalele din gradina zoo, in functie de necesitatile acestora.

2.2 Functiile Produsului

Aplicatia web managerieaza informatiile despre animalele dintr-o gradina zoologica. Despre aceste animale sunt informatii precum : denumire populara/stiintifica, descriere, origine, statut, caracteristici, dependenta de clima, periculozitate, proprietati speciale, specii inrudite, dusmani naturali, imagini de interes, regim alimentar, mod de inmultire.

Pagina Principala : Pagina din care pot fi accesate celelalte pagini ale website-ului

  • sign up : redirectioneaza catre sign up page.

2.3 Clasele și caracteristicile utilizatorilor

  • User : user asociat cu un cont poate utiliza facilitatile website-ului.
  • Admin : are optiunea de a manageria aspecte diferite ale website-ului.

2.4 Mediul de operare

Pagina web poate fi utilizata pe orice device care are o conexiune la internet si un browser instalat.

2.5 Constrângeri de proiectare și implementare

Zoo WebPage va fi dezvoltat in Php, si va folosi html, css si javascript pentru interfata.

4. System Features

4.1 Home page

Guests and users can read a little about the motivation behind the web app and navigate to all the other pages from this page.

4.2 Login page

Utilizatorii aplicației se pot autentifica introducândinformatiile corecte și, dupa validarea, accesează alte funcționalități ale aplicației.

4.3 SignUp page

Oaspeții își pot crea un cont și apoi îl pot folosi pentru a se autentifica în aplicație și a accesa toate caracteristicile oferite de aplicatie.

4.4 User Main page

Utilizatorii pot cauta diferite animale in functie de alegerile lor al informatiilor dupa care cauta respectivele animale din gradina zoologica. Poate vedea diferite informatii in functie de ce doreste acesta.

4.5 About us page

Utilizatori conectati sau nu pot observa informatiile despre pagina noastra si despre gradina zoologica.
